# 使用ShellTool进行数据分析

## Python数据分析示例

```bash
# 使用Python进行数据分析
python3 << 'EOF'
import pandas as pd
import numpy as np

# 读取数据
df = pd.read_csv('data.csv')

# 描述性统计
print(df.describe())

# 分组统计
print(df.groupby('category').agg({
    'value': ['mean', 'sum', 'count']
}))

# 相关性分析
print(df.corr())
EOF
```

## SQL查询示例

```bash
# 使用SQL进行数据查询
sqlite3 data.db << 'EOF'
-- 基础统计
SELECT 
    category,
    COUNT(*) as count,
    AVG(value) as avg_value,
    SUM(value) as total
FROM records
GROUP BY category
ORDER BY total DESC;

-- 趋势分析
SELECT 
    date,
    SUM(amount) as daily_total
FROM transactions
GROUP BY date
ORDER BY date;
EOF
```
