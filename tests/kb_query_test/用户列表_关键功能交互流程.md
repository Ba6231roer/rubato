# 用户列表 关键功能交互流程

## 流程1：新增用户
**前置条件**：已登录系统，进入用户列表页面
**步骤**：
1. 点击"新增用户"按钮 → [元素: e15, 操作: click]
2. 弹出新增用户弹窗
3. 填写用户名 → [元素: e20, 操作: fill]
4. 填写邮箱 → [元素: e21, 操作: fill]
5. 选择角色 → [元素: e22, 操作: select]
6. 点击"确定"按钮 → [元素: e25, 操作: click]
**预期结果**：新增用户成功，列表刷新显示新用户
**注意事项**：用户名不能重复

## 流程2：删除用户
**前置条件**：已登录系统，用户列表中有数据
**步骤**：
1. 点击某用户的删除按钮 → [元素: e30, 操作: click]
2. 弹出确认对话框
3. 点击"确认删除" → [元素: e35, 操作: click]
**预期结果**：用户被删除，列表刷新
**注意事项**：管理员账户不可删除

## 流程3：搜索用户
**前置条件**：已登录系统，进入用户列表页面
**步骤**：
1. 在搜索框输入关键词 → [元素: e10, 操作: fill]
2. 点击搜索按钮 → [元素: e11, 操作: click]
**预期结果**：列表显示匹配的用户
**注意事项**：支持模糊搜索
